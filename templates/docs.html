The Animal API

See interesting images and information about various animals...

This API allows you to search and get random information and images of different animals for mostly entertainment purposes!

Main Endpoint

/api/v1/search/animal

Parameters

ParameterTypeRequiredDescription
namestringNoThe name of the animal you want to search for. If not provided, a random animal will be selected.

Response Format

{ "name": "lion", "url": "https://cdn.statically.io/gh/henrique-coder/theanimalapi-dataset/main/lion/lion-a1b2c3d4.avif", "timestamp": "2025-04-02T14:46:00Z" }
